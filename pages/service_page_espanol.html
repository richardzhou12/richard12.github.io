Proceso de Servicio

1. Identificación de tus necesidades: Revisa nuestra lista de productos para ver si ya tenemos lo que estás buscando. Si no es así, también podemos trabajar contigo para encontrar un producto adecuado.

2. Búsqueda del producto: Trabajamos con nuestras fábricas asociadas para encontrar un producto adecuado para ti a un precio reducido.

3. Informe de cotización: Creamos un informe de cotización que incluye los detalles del producto, el precio y el tiempo de entrega.

4. Revisión del informe de cotización: Repasamos el informe de cotización contigo y respondemos cualquier pregunta que puedas tener. Si estás satisfecho con el producto y el precio, pasamos al siguiente paso. Si no es así, buscaremos un mejor precio o un producto más adecuado.

5. Pago: Si estás satisfecho con el producto y el precio, solicitamos un depósito del 30% del precio total mediante transferencia bancaria para iniciar el proceso de producción.

6. Producción y verificación de calidad: Después de recibir el depósito, nuestra fábrica asociada comenzará a producir el producto. Una vez que el producto esté terminado, lo examinaremos para asegurarnos de que cumpla con tus requisitos.

7. Pago final y envío: Una vez que el producto haya pasado la verificación de calidad, te pediremos que pagues el 70% restante del precio total. Después de recibir el pago, te enviaremos el producto.

8. Seguimiento: Después de que el producto haya sido entregado, haremos un seguimiento contigo para asegurarnos de que estés satisfecho con el producto y el servicio.


Si tienes alguna pregunta sobre la información anterior, contáctanos.

Gracias por elegir nuestro servicio. Siempre estamos dispuestos a ayudarte a encontrar el producto adecuado al precio correcto.
